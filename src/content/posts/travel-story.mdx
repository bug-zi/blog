## 为什么选择 Next.js

Next.js 提供了出色的开发体验：

- **App Router**：基于文件的路由系统
- **SSG/SSR**：静态生成或服务端渲染
- **MDX 支持**：在 Markdown 中使用 React 组件

## 技术栈

```typescript
const techStack = {
  framework: "Next.js 15",
  styling: "TailwindCSS v4",
  content: "MDX",
  deploy: "Vercel",
};
```
